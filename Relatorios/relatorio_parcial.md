## Introdução

Este estudo analisa características de repositórios populares do GitHub, utilizando dados coletados da API GraphQL.

Foram analisados os 1000 repositórios com maior número de estrelas na plataforma.

## Hipóteses

RQ1: Repositórios mais populares tendem a ser mais antigos.

RQ2: Repositórios populares possuem grande número de pull requests aceitos.

RQ3: Repositórios populares tendem a publicar releases com frequência.

RQ4: Repositórios populares são atualizados frequentemente.

RQ5: Algumas linguagens dominam entre os repositórios mais populares.

RQ6: Repositórios populares possuem alta proporção de issues fechadas.

## Metodologia

Os dados foram coletados utilizando a API GraphQL do GitHub, selecionando os 1000 repositórios com maior número de estrelas.

Os dados foram armazenados em JSON e posteriormente convertidos para CSV para análise estatística.
